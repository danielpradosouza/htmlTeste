Vamos falar sobre cores

Cores Hexadecimais

a seguir uma lista de cores Hexadecimais

  1. O modelo hexadecimal representa cores como uma combinação de valores de vermelho (R), verde (G) e azul (B), em formato hexadecimal (base 16)
  2. As cores hexadecimais funcionam como um código de seis dígitos, precedido por um #, que define a intensidade de vermelho, verde e azul (RGB) em um monitor. Os dois primeiros dígitos representam o vermelho, os dois seguintes o verde, e os dois últimos o azul. Cada par de dígitos varia de 00 (sem intensidade) a FF (intensidade máxima), e a combinação dessas três cores resulta em uma cor final.
  3. exemplos
    1. #FFFFFF: Vermelho $FF$ + Verde $FF$ + Azul $FF$ = Branco (intensidade máxima de todas as cores).
    2. #FF0000: Vermelho $FF$ + Verde $00$ + Azul $00$ = Vermelho puro (vermelho máximo, sem verde e azul)
    3. #00FF00: Vermelho $00$ + Verde $FF$ + Azul $00$ = Verde puro.

Cores RGB

a seguir uma lista de cores RGB

  • Em RGB, a cor "red" significa vermelho e é um dos três componentes primários de cores aditivas usados em dispositivos eletrônicos.
  • No sistema RGB (Red, Green, Blue), Green (Verde) é uma das três cores primárias de luz usadas para criar outras cores em dispositivos eletrônicos como monitores e TVs.
  • No modelo RGB (Red, Green, Blue), blue (azul) é uma das três cores primárias aditivas, e quando o valor de "blue" é 255, enquanto "red" e "green" são 0, a cor resultante é um azul puro (\(0,0,255\)). O sistema RGB é usado em dispositivos eletrônicos como TVs e monitores para reproduzir cores através da combinação de luz.
